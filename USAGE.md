# Using this responsibly

This exists for one reason: so you can **self-learn AI fluency**. The real
engagement here is *learning* — not reselling. If that's why you're here, welcome.

## You're welcome to
- Clone it and run it locally
- Read the code, learn how it works, adapt it for your own learning
- Share it and build on it — under the same non-commercial terms, with credit

## Please don't
- Sell it, or fold it into a commercial product or paid service
- Strip the attribution / `Required Notice`

## Clone & run
```sh
git clone https://github.com/anandsureshworks/anandsureshworks-com.git
cd anandsureshworks-com
python3 -m http.server 8000   # then open http://localhost:8000
```
It's a static site — no build step, no dependencies. Each tool runs entirely in
your browser.
