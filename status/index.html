Fleet discipline

Deterministic gates on the machine that builds all of this — re-run and republished daily as aggregates (counts only; details never leave the host).

How this works. Detection is deterministic and cheap — generated_at freshness contracts, an RCA mechanism health gate, dirty‑tree audits, launchd liveness checks. Agents are reserved for diagnosis when a sensor trips, never for detection: a watcher that can fail silently is the exact failure class this fleet exists to kill. Every incident that survives to a root cause becomes a runnable mechanism test, and those tests are what the "RCA mechanisms" count re-verifies daily.
